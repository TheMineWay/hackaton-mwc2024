# Hackathon-MWC2024 - Team 11

## Track Me To The Moon

Maintain constant communication through your devices in lunar territory. The goal of this application is to provide users with the ability to monitor and manage their devices remotely ensuring connectivity. The user will be able to have their devices centralized in a single app.

### General Description

This Remote Tracking app allows users to remotely track their devices, connect them to their PC or mobile device and make video calls, check their status and check their online connectivity.

## API

In order to make that possible, the app uses diferents API's: 

- [Device Location](https://developer.networkascode.nokia.io/docs/location/location-verification)
- [Device Status](https://developer.networkascode.nokia.io/docs/status/device-connectivity)
- [Network Slicing](https://developer.networkascode.nokia.io/docs/slicing/slice-creation)

## Installation

Clone the repository

```Bash
git clone https://github.com/TheMineWay/hackaton-mwc2024
```

Access to front directory for install the packages

```Bash
cd front
npm i
```

Start Vite

```Bash
npm run dev
```

Set up environment variables

```Bash
create a .env file in the backend folder and define the API Key
```

Meanwhile, open another console and access to server folder, install the packages and run the server

```Bash
cd ../back
npm i
```

```Bash
npm run start:dev
```
